# Survey Studio Demo Survey Bank

This file contains 20 demo surveys for your Survey Studio application. The set includes fun surveys for demos, plus practical business and workplace surveys. Each survey has 8 to 12 questions and uses the question types visible in the app: multiple choice, short text, long text, and rating.

For multiple choice questions, paste the options as comma-separated values into the Options field.
